Практическое задание

Написать консольное приложение Task Manager, которое выводит на экран запущенные процессы и позволЯет завершить указанный процесс. Џредусмотреть возможность завершениЯ процессов с помощью указаниЯ его ID или имени процесса. ‚ качестве примера можно использовать консольные утилиты Windows tasklist и taskkill.


Задание:

Приложение выводит на экран список процессов в таблице, количество процессов, отображающихсЯ одновременно на экране задаетсЯ либо константой в коде, либо параметром командной строки
* Клавиши 1 и 2 - Џромотать вверх-вниз 
* Клавиши 3 и 4 - промотать вверх-вниз постранично
* Клавиша 5 - обновить список
* Клавиш 6 и 7 - завершить нужный процесс введЯ его ID 
* -h - вывод минисправки по параметрам командной строк 
* -l <число>, задает количество одновременно отображаемых процессво на экране, если 0 или не задано, то выводитсЯ весь список.
